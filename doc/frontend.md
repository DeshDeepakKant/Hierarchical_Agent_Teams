# Frontend Documentation

The frontend of Hierarchical Agent Teams is a modern, reactive web interface built with **Vue 3**.

## Technology Stack
- **Framework**: Vue 3 (Composition API)
- **UI Components**: Single File Components (`.vue`)
- **Communication**: WebSockets for real-time streaming
- **Styling**: Responsive CSS with theme support

## User Interface Sections

### 1. Control Panel
Allows users to input their research queries and select the orchestration graph type. It features a "Send" button to initiate agent collaboration.

### 2. File Manager
Displays a real-time list of files generated by the agents in the workspace.
- **Refresh**: Manually sync the list with the backend.
- **Download**: Click on any file to download it to your local machine.

### 3. Collaboration Stream (Flow Area)
A dynamic message flow that shows the step-by-step progress of the agent teams.
- **Agent Avatars**: Differentiates between various agents (Search, Supervisor, Writer, etc.).
- **Real-time Rendering**: Messages appear as they are being generated by the AI models.

### 4. Theme Support
Includes a built-in dark/light mode toggle to enhance user experience during long research sessions.

## Local Development
To run the frontend locally:
```bash
cd frontend/agent-teams-frontend
npm install
npm run serve
```
The application will typically be available at `http://localhost:8080`.
